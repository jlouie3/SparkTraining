# SparkTraining
Sandbox for learning Spark 2.1 using resources from Pluralsight and Learning Spark (O'Reilly)

##### Environment: 
* Intellij IDEA
* Maven
* Spark 2.1
* Scala 2.11


##### For Windows
1. Create HADOOP_HOME environment variable
2. Add to path
3. Download and add winutils.exe from Hortonworks to HADOOP_HOME directory
